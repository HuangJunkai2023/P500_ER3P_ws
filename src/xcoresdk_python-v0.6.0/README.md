# xCore SDK 机器人控制接口

* xCore SDK编程接口库是珞石机器人提供给客户用于二次开发的软件产品.
* 新版 python sdk

## 兼容性

### 机器人控制器

* xCore V2.2.0及以上版本

### 编译环境

|操作平台|解释器|平台|语言|
|----|---|----|----|
|Ubuntu 18.04/20.04/22.04|python3.8.X及以上，默认3.10|x86_64|python|
|Windows 10|python3.8.X及以上，默认3.12|x86_64|python|

### 使用示例

**注意**: 应将库文件所在路径添加至运行脚本可识别的路径中。将脚本中的ip修改为连接机器人所设置的ip。
见example文件夹中的示例代码。
